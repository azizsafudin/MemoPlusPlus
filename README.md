# MemoPlusPlus
Simple extension for [memo.cash](https://memo.cash/profile/1HezZbHLhd6fcKs1ytUxDjSF3mcUdHiUeL) for my own needs.

### Features (so far)
1. Ability to mute posts from certain users.
2. That's it

### Installation
1. Git clone
2. Drag and drop folder into chrome://extensions
